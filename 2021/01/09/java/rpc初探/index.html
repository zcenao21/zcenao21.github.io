RPC

RPC(Remote Procedure Call)，即远程过程调用，主要用于分布式情景下节点间的调用，可以让远程调用使用起来就像本地调用一样方便。

最简单RPC实现包括如下内容

  • 调用者发起调用
  • 封装调用，打包方法名和参数，序列化
  • 被调用端反序列化，得到方法名参数，使用反射并执行得到结果
  • 将结果序列化返回给调用者
  • 调用者反序列化得到结果
